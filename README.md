[Daily city insights]

- Description: It provides the user with today's info regarding a city that the user inputs its name, that includes: weather, population, and the latest news related to this city or the country itself. It also provide a fun fact of the day.

- Add your own API keys to replace the API_Keys.env file. It should contain the following:
```
weather = "your_weatherapi_API_key"
news = "your_gnews_API_key"
pop = "your_geonames_username"
```
